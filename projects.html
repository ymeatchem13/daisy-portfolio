Projects

Grocery Store App

In this project, I worked individually to create a mobile app for a grocery store application for a client. With a focus on user convenience, I crafted an intuitive app where users can browse and make purchases from various convenience stores right on their phones. While ensuring seamless interaction and intuitive navigation, I created wireframes that turned into functional frames. Within this app, I implemented responsive buttons, intuitive dropdown menus, and clickable navigation elements to make using this app effortless.

UC Alert Maps

In this project, I collaborated with a team to develop a mobile design based on the UC Alert Maps website. As the UX Designer, I created wireframes that were transformed into functional map frames, enabling users to pinpoint their location while using the application. Additionally, I designed the homepage, mission statement, and implemented various functional aspects of the website, such as buttons, dropdown menus, and clickable navigation elements.
